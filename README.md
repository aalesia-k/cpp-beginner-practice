# C++ Beginner Practice Programs

This repository contains foundational C++ programs developed to reinforce core programming concepts including user input handling, arithmetic operations, and structured logic.

## Projects Included

### 1. Donut Shop Order Simulation
A console-based ordering system that calculates pricing and tax for donut purchases.

Features:
- User input collection (name, phone number, order quantity)
- Dozen and individual item price calculations
- Tax computation
- Currency formatting

Concepts Used:
- Input/output streams
- Arithmetic calculations
- Modulo and division operations
- Precision formatting

---

### 2. Minutes to Hours Converter
Converts a total number of minutes into hours and remaining minutes.

Features:
- Integer division and modulo operations
- User interaction
- Output formatting

Concepts Used:
- Arithmetic operators
- Integer math
- Console I/O

---

### 3. Field Goal Percentage Calculator
Computes shooting percentages for individual athletes and team totals.

Features:
- Percentage calculations
- Precision control
- Statistical computation

Concepts Used:
- Floating-point arithmetic
- Type casting
- Formatted output

---

## Technical Focus

- Core C++ syntax
- Control structures
- Arithmetic logic
- Input validation basics
- Foundational problem-solving

## Purpose

These projects represent early-stage development work and demonstrate progression toward more structured and data-driven applications.
